### 下载安装
- 获取项目代码:  
``` xml
git clone https://gitee.com/luo-derong/vue3-shop-admin.git
```
- 安装依赖:  
``` xml
cd vue-shop-admin
yarn install
```
- 运行:  
``` xml
yarn dev
```
- 打包:  
``` xml
yarn build
```
